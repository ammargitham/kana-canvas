# KanaCanvas

KanaCanvas is an easy to use web app to improve your Japanese handwriting with easy-to-follow stroke orders and real examples of proper handwriting.

## Running locally

```bash
npm run dev
```

## Preview site

```bash
npm run preview
```

## Deploy to Cloudflare

```bash
npm run deploy
```
